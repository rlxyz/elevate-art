# Introduction

##### ELEVATE.ART: The Art Compiler

---

ELEVATE.ART is a powerful generative art compiler which anyone can permissionlessly use to upload art layers to generate and mint a large scale NFT collection to sell.

Learn more about the lastest version of ELEVATE.ART [here.](../docs/v3-overview)

#### ELEVATE.ART Stack

- [Smart Contracts:](../docs/smart-contracts/zora-v3) ELEVATE.ART deployed smart contracts 

- [API:](../docs/zora-api/intro) Retrieve NFT metadata and market data on any ERC-721

- [ZDK:](../docs/zora-api/zdk) A development kit for interacting with ZORA's API 

- [NFT Hooks:](../docs/developer-tools/nft-hooks/intro) The simplest way to interact with ZORA's API 

- [NFT Rendering:](../docs/developer-tools/nft-rendering/introduction) Tools for easily displaying NFT data
