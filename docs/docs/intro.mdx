# Introduction

##### ZORA: The NFT Marketplace Protocol

---

ZORA is a decentralized protocol where anyone can permissionlessly buy and sell NFTs.
We've created a series of tools that makes it easy to get started building. Learn more about the lastest version of ZORA [here.](../docs/v3-overview)

#### ZORA Stack

- [Smart Contracts:](../docs/smart-contracts/zora-v3) ZORA's deployed smart contracts 

- [API:](../docs/zora-api/intro) Retrieve NFT metadata and market data on any ERC-721

- [ZDK:](../docs/zora-api/zdk) A development kit for interacting with ZORA's API 

- [NFT Hooks:](../docs/developer-tools/nft-hooks/intro) The simplest way to interact with ZORA's API 

- [NFT Rendering:](../docs/developer-tools/nft-rendering/introduction) Tools for easily displaying NFT data
