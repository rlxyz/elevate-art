# Elevate.Art: The Leading Art Generator

---

Elevate.Art is a decentralized protocol where anyone can permissionlessly buy and sell NFTs.
We've created a series of tools that makes it easy to get started building. Learn more about the lastest version of Elevate.Art [here.](../docs/v3-overview)

#### Elevate.Art Stack

- [Smart Contracts:](../docs/smart-contracts/Elevate.Art-v3) Elevate.Art's deployed smart contracts 

- [API:](../docs/Elevate.Art-api/intro) Retrieve NFT metadata and market data on any ERC-721

- [ZDK:](../docs/Elevate.Art-api/zdk) A development kit for interacting with Elevate.Art's API 

- [NFT Hooks:](../docs/developer-tools/nft-hooks/intro) The simplest way to interact with Elevate.Art's API 

- [NFT Rendering:](../docs/developer-tools/nft-rendering/introduction) Tools for easily displaying NFT data
